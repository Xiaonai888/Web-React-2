# Shadow App — React

## 📁 Structure

```
src/
├── components/
│   └── Footer.jsx     ← ✅ កែត្រង់នេះ — Footer ផ្លាស់ប្តូរគ្រប់ page!
├── pages/
│   ├── ForYou.jsx     ← / (For You)
│   ├── Fast.jsx       ← /fast
│   ├── Discover.jsx   ← /discover
│   ├── Library.jsx    ← /library
│   ├── Me.jsx         ← /me
│   └── Search.jsx     ← /search
├── App.jsx            ← Router
└── main.jsx
```

## 🚀 Run

```bash
npm install
npm run dev
```

Open: http://localhost:5173

## 📦 Deploy to Vercel

```bash
git add . && git commit -m "update" && git push
```

Vercel auto-deploys!
